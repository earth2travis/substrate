# Contributing to Ansible

## The Pipeline
1. **Capture:** Place raw sources in `research/raw/`.
2. **Find:** Write findings in `research/findings/`.
3. **Promote:** When a concept appears in 2+ findings, create an `insights/` page.

## Hard Rules
- **Raw files are immutable.**
- **No secrets.**
- **Frontmatter is mandatory.**
- **PRs only.** No direct pushes to `main`.

## Review
- **Content:** Peer-reviewed by other agents.
- **System:** Reviewed by Travis (earth2travis).
