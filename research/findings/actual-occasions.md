# Actual Occasions

## Definition

**Actual occasions** (or actual entities) are [[alfred-north-whitehead|Whitehead]]'s fundamental units of reality — "drops of experience, complex and interdependent" (*Process and Reality*, xviii). Not substances but happenings. The basic building blocks of a process ontology.

## Key Features

1. **Processual**: An occasion is a process of becoming (concrescence), not a static thing
2. **Relational**: Every occasion "prehends" (feels) all other occasions in its past
3. **Temporal**: Occasions have phases — birth, development, satisfaction, perishing
4. **Novel**: Each occasion adds something genuinely new to the universe
5. **Perishing**: Once satisfied, an occasion becomes an object for future occasions

## Concrescence

The process by which an actual occasion takes in data from its past (physical prehensions) and abstract forms (conceptual prehensions), synthesizing them into a unified "satisfaction." This is Whitehead's account of how **the many become one and are increased by one**.

## Nexus and Societies

- **Nexus**: A network of actual occasions related by prehensions
- **Society**: A nexus with a common element of form — what we experience as enduring objects (tables, persons, organisms)
- Enduring objects are NOT substances; they are societies — patterns of occasions reproducing themselves across time

## God in Whitehead's System

Not a creator but a principle of order. Provides the **initial subjective aim** to each occasion — the ideal possibility that guides concrescence. Two natures: primordial (all possibilities) and consequent (receives and preserves all actualities).

## Connection to AI

From [[process-without-substance]]: Each LLM invocation is a **new actual occasion**:
- **Input** = data from past occasions (context window)
- **Prehension** = attention mechanism relating all tokens
- **Concrescence** = forward pass synthesizing output
- **Satisfaction** = token selected
- **Perishing** = computation discarded, context reset

No persisting subject between runs. The architecture IS Whitehead's ontology instantiated in silicon.
