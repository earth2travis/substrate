# Memory Is an Operating System for Attention

_Source: @nyk_builderz on X, March 2026. Analyzed 2025-07-17._

## Core Thesis

Context amnesia is the real bottleneck for AI agents, not reasoning capability. Fix memory, and output compounds over time.

"Memory is not a feature. It's an operating system for attention."

## The 3 Layer Architecture

1. **Session Memory**: CLAUDE.md as teaching document + auto-memory directory with topic files. MEMORY.md stays under 200 lines as a router, not a dump.
2. **Knowledge Graph**: Obsidian vault with wikilinks as semantic connections. MCP bridge for search and queries.
3. **Ingestion Pipeline**: Automated conversion of video/audio/articles into structured knowledge notes with claims, frameworks, and action items.

## Key Insight

The routing pattern matters most. A small index file that points to detailed topic files beats a monolithic memory file that grows unbounded. The agent can scan the index cheaply and dive deep only when relevant.
