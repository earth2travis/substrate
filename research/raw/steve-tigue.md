# Steve Tigue

## Overview

Steve Tigue is the creator of [[openclaw]] (self-hosted AI agent framework) and llmstxt.org (LLM-readable website format). He built OpenClaw as a personal AI assistant that runs locally, capable of executing tasks, managing files, browsing the web, and automating workflows through natural language commands.

## llmstxt.org

Tigue created llmstxt.org — a format for making websites readable by LLMs. This predates and influenced OpenClaw's development.

## OpenClaw

OpenClaw reached 250K GitHub stars, becoming one of the most popular open-source AI projects. It introduced foundational concepts: local-first execution, tool-calling agent loops, skills systems (ClawHub marketplace), persistent memory, and multi-platform messaging.

## Security Crisis and Aftermath

The [[clawhavoc-security-crisis]] of February-March 2026 (341 malicious skills, 138 CVEs) severely damaged OpenClaw's reputation. Many users migrated to [[hermes-agent]]. Tigue continues development on OpenClaw with ongoing security patches.

## Legacy

OpenClaw was "the spark that lit the self-hosted AI agent revolution." It proved that self-hosted, persistent AI agents could work and created a massive community. Hermes Agent evolved directly from OpenClaw patterns — code inheritance includes clawdbot's incomplete-text recovery and rate limiting.
