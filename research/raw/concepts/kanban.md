# Kanban

> Visual system for managing work and controlling inventory in production, originating from the Toyota Production System. The signaling mechanism that enables [[just-in-time]] production.

## Meaning

- **Kanban** (看板) = "signboard" or "visual card" in Japanese
- A physical or digital card that signals when to produce or reorder materials

## How It Works

1. Each workstation has input and output areas
2. When output drops below a threshold, a kanban card signals the upstream station
3. The upstream station produces only enough to replenish what was consumed
4. This creates a pull system where production is triggered by actual demand

## Evolution

- Originally physical cards at Toyota
- Evolved into software tools (Kanban boards) for project management
- Used in lean manufacturing, agile software development, and knowledge work
- Digital tools: Trello, Jira, Azure DevOps

## Benefits

- Visual transparency of work and bottlenecks
- Limits work-in-progress (WIP)
- Prevents overproduction
- Self-regulating inventory system
- Adaptable to any type of work
