# DevOps

Culture and set of practices combining software development (Dev) and IT operations (Ops). Shortens the development lifecycle while delivering features, fixes, and updates in close alignment with business objectives.

## Connection to Lean Software

DevOps shares lean's emphasis on flow, feedback loops, and continuous improvement. The [[dora-metrics]] (deployment frequency, lead time, change failure rate, restore time) originated from DevOps research and are the primary quantitative measures of software delivery performance.

## Key Practices

- Infrastructure as code (IaC)
- Continuous integration and [[continuous-delivery]]
- Monitoring and observability
- Automated testing
- Collaboration between development and operations
