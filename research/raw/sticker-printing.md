# Sticker Printing Research

_February 8, 2026_

Research for printing YC mission patch stickers. Quality is the priority.

---

## TL;DR Recommendation

**Primary: Sticker Mule** (industry standard, proven quality)
**Alternative 1: Sticker Giant** (scored backing, B Corp, US-based)
**Alternative 2: Sticker Ninja** (festival-tested durability, premium materials)

---

## 1. Sticker Mule

**Website:** stickermule.com

### File Requirements

| Requirement    | Specification                         |
| -------------- | ------------------------------------- |
| Resolution     | 300 PPI minimum (vector preferred)    |
| Formats        | AI, PDF, EPS, SVG, PNG, JPG, PSD, TIF |
| Bleed          | 1/16" (0.0625") minimum               |
| Border         | Standard 1/10", minimum 1/16"         |
| Font size      | 6pt minimum                           |
| Line thickness | 1pt minimum                           |
| Color mode     | CMYK recommended                      |

### Quality

- Thick, durable vinyl
- Scratch resistant
- Waterproof (rain and sunlight protected)
- Matte or glossy finish available
- Die cut to any shape

### Pricing (Die Cut)

- 50 stickers (3"): ~$72
- Free shipping (4 day)
- $1 sample pack available

### API

- **No public API for orders** (feature requested on their Canny board)
- They use Contentful internally but don't expose ordering API

### Crypto

- **Unknown** (no evidence of crypto acceptance found)

### Notes

- Industry standard quality
- Fast turnaround (4 days typical)
- Free online proofing
- Some users boycott due to owner's political views

## 2. Sticker Giant

**Website:** stickergiant.com
**Location:** Longmont, Colorado (US-based)

### File Requirements

| Requirement | Specification                            |
| ----------- | ---------------------------------------- |
| Resolution  | 300 DPI at final size (vector preferred) |
| Formats     | AI, PDF, EPS, PNG, JPG, PSD              |
| Color mode  | CMYK                                     |

### Quality

- Premium vinyl material
- **Scored backing** (easier to peel, unique feature)
- Waterproof and UV resistant
- In-house art team reviews every file
- Certified B Corporation

### Pricing (Die Cut)

- 250 stickers (2"): ~$117
- Higher than Sticker Mule but includes art review

### Special Options

- Glow in the dark
- Holographic
- Clear vinyl
- Glitter finish

### API

- **No public API**

### Crypto

- **Unknown**

### Notes

- Home of "Saul the Largest Sticker Ball" (Guinness World Record)
- In business since 2000
- Strong reputation for quality and customer service
- Slightly higher price point but premium experience

## Preparing Our File

Our mission patch (shield-v3a.png) needs conversion for print:

### Current State

- Format: PNG
- Resolution: High (Gemini output)
- Size: ~921 KB

### Recommended Steps

1. **Convert to vector (optional but ideal)**
   - Use Adobe Illustrator's Image Trace
   - Or Vectorizer.ai for automatic conversion
   - Preserves quality at any size

2. **Set up for die cut**
   - Add 1/16" bleed beyond cut line
   - Ensure CMYK color mode
   - Save as PDF or AI

3. **Specify size**
   - Recommend 3" diameter for visibility
   - Could also do 2" for laptop stickers

4. **Order proof**
   - All three vendors offer proofing
   - Review colors before full production
